# 🎓 Training Initiative — تطبيق السحب العشوائي

تطبيق ويب تفاعلي مبني بـ React يُستخدم في الدورات التدريبية لتسجيل أسماء المشاركين وسحب اسم عشوائي منهم.

---

## 🚀 تشغيل المشروع

```bash
# 1. تثبيت الحزم
npm install

# 2. تشغيل بيئة التطوير
npm run dev
```

ثم افتح المتصفح على: `http://localhost:5173`

---

## 🗂️ هيكل المشروع

```
src/
├── context/
│   └── NamesContext.jsx    # إدارة الحالة العامة + حفظ البيانات في localStorage
├── components/
│   └── Navbar.jsx          # شريط التنقل المشترك
├── pages/
│   ├── InputPage.jsx       # صفحة تسجيل الأسماء  →  /
│   └── AdminPage.jsx       # صفحة السحب العشوائي →  /admin
├── App.jsx                 # إعداد التوجيه (React Router)
└── index.css               # Tailwind CSS + الأنيميشن المخصص
```

---

## 📄 الصفحات

### صفحة التسجيل `/`
- يكتب المشارك اسمه في حقل الإدخال
- يضغط "Register Name" لإضافة الاسم
- تظهر رسالة تأكيد فورية عند الإضافة بنجاح
- عداد يُظهر عدد المسجّلين الحاليين مع رابط للانتقال لصفحة الإدارة

### صفحة الإدارة `/admin`
- تعرض قائمة كاملة بجميع الأسماء المسجّلة مع أرقامها
- زر **"Pick a Random Name"** يسحب اسمًا عشوائيًا
- يظهر الفائز في بطاقة بارزة مع رقمه وتأثير بصري مميز
- الاسم الفائز يُميَّز داخل القائمة بلون مختلف وأيقونة نجمة
- زر لمسح جميع الأسماء مع رسالة تأكيد

---

## 💾 حفظ البيانات

لا يوجد backend. الأسماء تُحفظ في `localStorage` تحت المفتاح `training_names`، مما يعني:
- تبقى الأسماء محفوظة عند إغلاق المتصفح وإعادة فتحه
- البيانات خاصة بالمتصفح والجهاز المستخدَم
- يمكن مسحها من صفحة الإدارة أو من إعدادات المتصفح
